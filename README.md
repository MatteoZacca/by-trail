# by-trail

## Setup and Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start MongoDB server and Express application using Docker Compose:

    ```bash
    docker compose up -d
    ```

3. Check the database structure with Mongo Express at [http://localhost:8081](http://localhost:8081).

4. Open a new terminal inside the project directory and run the Express server:

   ```bash
   node .
   ```
